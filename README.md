# Instructions
- The data is already organized such that the matlab files are not needed, but are left to display preprocessing
   - If you'd like to generate the RDMs for yourself then refer to the [data's authors](https://github.com/ci4r/CI4R-MULTI3/blob/master/README.md)

- 1) ensure that the file order from the Repo is the same as downloaded
- 2) install all the dependent libraries from the "requirements.txt" within scripts
- 3) from within either CNN_manual.py, CNN_pytorchImp.py, MLP_manual.py modify the "#%% Paths" related cell to the data set you'd like to use.
